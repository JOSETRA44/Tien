Tien.
Android Productivity App

Productividad Escalable y Nativa

Descubre Tien, la aplicación centralizada para Notas y Agenda, construida con Jetpack Compose, Material 3 y SQLite nativo en C++.

Capacidades Clave

Una única experiencia, diseñada para eficiencia y control constante de tus tareas y notas.

📝

Notas

Captura rápida de información, edición fluida, organización mediante búsqueda y orden por recientes, antiguas o alfabético. Posibilidad de recuperar notas eliminadas.

📅

Agenda

Gestión de tareas con fecha de vencimiento, prioridad (baja, media, alta) y control de vida de la tarea. Filtrado por día dentro de la aplicación.

🎨

UX / UI

Implementación fiel de Material 3 con tema global claro/oscuro que cambia en tiempo real. Navegación fluida y retroalimentación proactiva al usuario.

Arquitectura & Capa de Datos

Separación de responsabilidades clara entre la UI, estado y persistencia mediante contratos de JNI estables.

Flujo Unidireccional

UI → ViewModel → NativeLib → DB → ViewModel → UI

  • Single source of truth en NotesUiState.
  • Lógica de negocio encapsulada y orquestada por el ViewModel.
  • Operaciones transaccionales y rápidas usando C++/SQLite, vinculadas vía JNI.
